## 🛠️ Technology Stack
* **Cloud Platform:** Microsoft Azure
* **Data Orchestration:** Azure Data Factory (ADF)
* **Data Lake:** ADLS Gen2
* **Data Warehouse:** Azure Synapse Analytics
* **Data Visualization:** Power BI Desktop & Service
